# Tplot

Tplot is text base graph plot tool.

## Installation

Add this line to your application's Gemfile:

    gem 'tplot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tplot

## Usage

List useful plugin list

    $ tplot list
